Claude Code + ChatGLM 全平台配置指南

概述

本文档介绍如何在Windows、macOS和Ubuntu系统上配置Claude Code使用ChatGLM（智谱AI）模型。

重要说明

智谱AI提供了Anthropic兼容的API端点，推荐使用以下配置方式：

  • API端点：https://open.bigmodel.cn/api/anthropic
  • 配置方式：使用 env 对象包装环境变量
  • 模型映射：支持Claude模型名称到GLM模型的自动映射

准备工作

获取智谱AI API Key

  1. 访问 智谱AI开放平台
  2. 注册并登录账号
  3. 在控制台获取API Key

关于智谱Coding Plan

智谱AI最近推出了Coding Plan服务，专门为代码生成和编程任务优化：

  • 默认模型：已升级至GLM-4.6
  • 特点：针对编程场景进行了专门优化
  • 兼容性：完全兼容Claude Code，无需额外配置
  • 优势：更好的代码理解、生成和调试能力

优惠活动

智谱AI目前提供以下优惠活动：

  • 首月尝鲜：新用户首月享受特别优惠价格
  • 拼车优惠：多人团购可享受更低折扣
  • 学生优惠：在校学生凭学生证可获得专属折扣
  • 企业方案：为企业提供定制化的优惠方案

注意：如果您之前在settings.json中配置过GLM-4.5的固定模型映射，建议更新为GLM-4.6以获得最新性能。

安装Node.js

所有平台都需要先安装Node.js（建议使用LTS版本）：

  • 官方下载：https://nodejs.org/
  • 或使用包管理器安装

    • 变量值：your_zhipu_api_key
    • 变量名：ANTHROPIC_BASE_URL
    • 变量值：https://open.bigmodel.cn/api/anthropic
    • 变量名：ANTHROPIC_DEFAULT_SONNET_MODEL
    • 变量值：glm-4.6

通用配置参数说明

settings.json 完整参数

参数说明

  • apiUrl: 智谱AI的API地址
  • authToken: 你的智谱API Key
  • model: 使用的模型（glm-4.6, glm-4-flash, glm-4-air）
  • maxTokens: 最大生成token数，建议8192
  • temperature: 温度参数，0-1之间，控制创造性
  • topP: 核采样参数，0-1之间
  • stream: 是否使用流式输出

常见问题

1. API地址配置

智谱AI的API地址为：

  • 生产环境：https://open.bigmodel.cn/api/paas/v4/chat/completions
  • 支持的模型：glm-4.6, glm-4-flash, glm-4-air

2. 模型选择建议

  • glm-4.6：最新版本，性能最好，适合复杂任务
  • glm-4-flash：响应速度快，适合简单任务
  • glm-4-air：成本更低，适合大批量任务

注意事项

  1. API Key保密

    • 不要将API Key提交到版本控制系统
    • 使用环境变量存储敏感信息
    • 定期轮换API Key

  2. 网络连接

    • 确保能访问智谱AI的API地址
    • 检查防火墙和代理设置
    • 考虑使用CDN加速

  3. 性能优化

    • 根据任务复杂度选择合适的模型
    • 调整maxTokens避免浪费
    • 使用流式输出提升体验

  4. 成本控制

    • 监控API使用量
    • 设置合理的请求频率限制
    • 选择适合的模型版本
